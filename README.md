# ETL INMET Avisos Meteorológicos

## Resumo

ETL dos avisos meteorológicos emitidos pelo Instituto Nacional de Meteorologia (INMET), disponibilizados por meio de um feed RSS no padrão CAP (Common Alerting Protocol) de acordo com o UF selecionado, na sequencia a geração do mapa com os municípios sob aviso de acordo com o evento, severidade, data/hora aviso.


Fonte dos dados:
**INMET – RSS de Avisos Meteorológicos**  
https://apiprevmet3.inmet.gov.br/avisos/rss
